      Download

      最左邊的是下載到 Ubique Panel 控制板的按鍵:

      連接usb並點擊download鍵即可將設計下載至控制板

      當下載成功時，ADE 會出現下面的彈出視窗:

      這表示整個 Design 已經成功下載到 Ubique Panel 的控制板上。下在時間會隨著設計使用圖片的多寡而變化，若設計專案有錯誤，視窗會顯示錯誤訊息，並停止下載。

      中間的按鍵是用來開啟內建的模擬器(Simulator):

      內建的模擬器是模擬使用者設計被載入Ubique Panel之後的顯示以及互動功能，使用者可以在個人電腦上以滑鼠操作設計出來的介面，模擬器會模擬完整圖介面功能。並且支援所見及所得 (What-You-See-Is-What-You-Get) 的特色，模擬的圖像與Ubique Panel上一致(色差可能發生於使用者使用的螢幕以及Ubique Panel之間，其色域規格以Ubique Panel規格書定義為準)。

      模擬器無法於個人電腦模擬Ubique Panel上的I/O, PWM,ADC等實體出入行為，但可以模擬Ubique Panel Register Communication Protocol。

      最右邊的按鍵啟動Messenger，目的是在個人電腦上與Ubique Panel或是模擬器通訊

      如果打開對外通訊的 Messenger，下面的視窗便會出現，

      在這視窗中，使用者可以透過個人電腦的COM Port，連結Ubique Panel，藉由Register communication protocol改變暫存器的內容值, 模擬由外部MCU透過UART1傳遞數值給Ubique Panel。

      以下詳細說明與模擬器以及Ubique Panel通訊方法:

      一.利用Messenger和模擬器通訊

      首先用ADE開啟一個既有設計

      確認IOConfig頁面UART1為Enabled且Baud Rate正確且暫存器已選取

      到Register Table 找到欲進行通訊的Register並勾選Sent to Host

      打開Messenger視窗與模擬器，進行以下操作

      1.設定Socket

      2.點擊連接，連接成功會看到彈出信息框顯示成功

      3.點擊連接確認，Messenger會送出連接確認命令，若是模擬器端正常接收，則會回復連接確認命令。

      4.連接正常，接收確認資訊ack

      5.通訊指令

      選擇Set Register 表示要寫入暫存器內容，並且設定暫存器ID以及要寫次的資料以及型態，該暫存器ID以及資料型態必須與ADE設計專案內暫存器設定相符。

      6.模擬器接收通訊指令且模擬器上會依通訊指令修改對應暫存器數值，若該暫存器有對應的Display Widget則模擬器進行相對應的顯示

      二. Messenger與Ubique Panel通訊

      首先利用ADE將既有的設計下載至博橡科技控制板上

      之後利用USB to TTL模組連接控制板RX1 TX1 GND與ADE所在電腦的USB

      連接完成後打開Messenger視窗

      1.設定Serial Port，利用[Scan]掃描更新串口，選擇正確的串口並設定鮑率

      2.點擊[Ack Test]確認連接

      3.連接正常，接收到ack訊息

      4.設定並傳送通訊指令

      接下來即可在Ubique Panel上看到相對應的顯示

      三. PC Simulator與Ubique Panel 通訊

      目的為快速測試設計運作是正常

      首先 利用ADE準備兩個設計，一個負責傳送數值，一個負責接收數值

      將負責傳送數值的設計利用模擬器打開，負責接收數值的設計下載至博橡科技控制板

      ※注意!!確保兩個設計欲溝通的暫存器ID相同，且傳送方要勾選[Send to Host]

      1.設定Serial Port，選擇正確的串口並且設定好鮑率

      2.利用Widget改變暫存器內容，再透過UART傳送至博橡控制板

      接下來可以在Ubique Panel上看到相對應的顯示
