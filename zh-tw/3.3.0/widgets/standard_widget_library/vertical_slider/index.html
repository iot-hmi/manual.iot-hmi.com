垂直條狀調節器可作為Output以及輸入元件，將Knob開啟時進度條為輸入元件，否則則為輸出元件。

垂直條狀調節器的背景、進度條、進度條底色和Knob等，皆可分別載入不同尺寸的圖形，使用者可以根據需求，製作不同風格及形狀的進度條，適合使用於需要顯示特別指示訊息的進度條，譬如電池存量或液體槽的液體高度等。
