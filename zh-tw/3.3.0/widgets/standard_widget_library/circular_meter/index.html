    Properties

    Circular Meter

    屬性名稱 屬性功能描述 操作型態
    Name Widget實體名稱 Name

    Visibility Control

    屬性名稱 屬性功能描述 操作型態
    Binding Hidden Control Register 決定是否要綁定暫存器，藉由暫存器內容觸發隱藏Widget Checkbox
    Hidden Control Register 透過改變暫存器值決定顯示或不顯示Widget Register Selector

    Behavior Setting

    屬性名稱 屬性功能描述 操作型態
    Minimum Data Value 對應到圓弧開始角度的資料值，必須小於Maximum Data Value Number Editor
    Maximum Data Value 對應到圓弧結束角度的資料值，必須大於Minimum Data Value Number Editor
    Start Angle 進度條圓弧開始角度(0-359)，詳情參考Number Editor Number Editor
    Arc Length 進度條圓弧長度(1-359) Number Editor
    Value Binding Register 暫存器的內容是Minimum Data Value的話，Circular Meter 會顯示底色，隨著暫存器數值增加，會顯示活躍顏色，直至暫存器是Maximum Data Value時Circular Meter顏色會顯示全滿 Register Selector

    Style Setting

    屬性名稱 屬性功能描述 操作型態
    Inactive Segment Color 設定未作動片段的顏色 Color Selector
    Active Segment Start Color 設定作動片段起始的主要顏色 Color Selector
    Active Segment End Color 設定作動片段結束的漸變顏色 Color Selector
    Tick Granularity 設定刻度的分格數 Number Editor
    Tick Width 設定刻度的寬度 Number Editor
    Tick Length 設定刻度的長度 Number Editor

    Preview

    屬性名稱 屬性功能描述 操作型態
    Show Protractor 是否要顯示輔助抓取點 Checkbox
    Value 將最小資料值與最大資料值對應圓弧角度和長度進行預覽顯示 Number Editor

    調整Widget的圓心 Center Editor
