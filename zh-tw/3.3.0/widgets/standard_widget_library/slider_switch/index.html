    滑動開關既是一個Input Widget也是Output Widget，主要作為一個雙態按鈕。

    狀態可以由使用者觸摸按鈕來改變，或是藉由其他Widget或是程式改變繫結的暫存器內容來改變按鈕狀態，利用這個特性，可以做到在不同頁面但是相互連動的雙切開關功能。

    按鈕的不同按鈕狀態可以對應到不同的顯示圖像以及文字。

    開關按紐時，會產生事件並可以對應到頁面切換、暫存器數值改變或是執行使用者撰寫的Blockly程式。

    屬性名稱 屬性功能描述 操作型態

    OFF State Text Setting

    屬性名稱 屬性功能描述 操作型態
    Display Text 決定開關為關時是否要顯示文字 Checkbox

    Text 顯示開關為關時的文字內容 Plain Text

    Color 設定開關為關時文字的顯示顏色 Color Selector
    Alignment 設定開關為關時文字的對齊方式 Alignment Selector
    Blending Mode 決定開關為關時文字的顏色與背景圖片的混色，詳情參考Blending Selector

    Switch ON Event Setting

    屬性名稱 屬性功能描述 操作型態

    Enable Register Operation 當按鈕被點擊時，根據Binding Register以及Operation Mode的內容對暫存器進行操作 Checkbox
    Binding Register 決定要改變的暫存器 Register Selector
    Operation Mode 當按鈕被點擊時，此處設定的行為會執行，詳情參考Register Operation Setting Register Operation Setting
    Enable Callback Function 當按鈕被點擊時，根據Behavior Programming設計的Callback Function進行操作 Checkbox
    Behavior Programming 當按鈕被點擊時，此處設定的行為會執行，詳情參考Blockly Editor Blockly Editor
