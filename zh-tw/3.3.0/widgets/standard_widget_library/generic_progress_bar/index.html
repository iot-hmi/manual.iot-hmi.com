    進度條可作為Output以及Input Widget，將Knob開啟時進度條為Input Widget，否則則為Output Widget。

    進度條可以有橫式以及直式兩種，視長寬的尺寸自行調整。此外進度條可以調整背景、進度條和Knob的邊框、顏色、透明度以及圓角半徑等風格屬性。

    當需要使用大量進條時，此Widget為兼顧美觀，效能與資源的最佳選擇。

    Properties

    Generic Progress Bar

    屬性名稱 屬性功能描述 操作型態
    Name Widget實體名稱 Name

    Visibility Control

    屬性名稱 屬性功能描述 操作型態
    Binding Hidden Control Register 決定是否要綁定暫存器，藉由暫存器內容觸發隱藏Widget Checkbox
    Hidden Control Register 透過改變暫存器值決定顯示或不顯示Widget Register Selector

    Behavior Setting

    屬性名稱 屬性功能描述 操作型態
    Minimum Data Value 最小資料值，對應到進度條開始 Number Editor
    Maximum Data Value 最大資料值，對應到進度條結束 Number Editor
    Value Binding Register 如果綁定的暫存器內容等於Minimum Data Value，Generic Progress Bar顯示在開始位置
    如果綁定的暫存器內容等於Maximum Data Value，Generic Progress Bar顯示在結束位置    		 Register Selector
    Preview Segment 選擇要預覽的段落狀態 Dropdown List

    Bar Line Style Setting

    屬性名稱 屬性功能描述 操作型態
    Main Color 設定進度條的主要顏色 Color Selector
    Gradient Color 設定進度條的漸變顏色 Color Selector
    Bar Opacity % 設定進度條的不透明度(0-100%) Number Editor
    Corner Radius 設定進度條的圓角半徑 Number Editor
    Border Color 設定進度條邊框的顏色 Color Selector
    Border Width 設定進度條邊框的寬度 Number Editor
    Shadow Color 設定進度條陰影的顏色 Color Selector
    Shadow Width 設定進度條陰影的寬度 Number Editor

    Segment 2 Style Setting

    屬性名稱 屬性功能描述 操作型態
    Enable Data Segment 2 決定是否要顯示第二段落 Checkbox
    Data Value 設定第二段落開始數值，當綁定的暫存器內容等於此數值時，顯示第二段落與其設定
    必須小於第三段落開始數值    		 Number Editor
    Main Color 設定第二段落的主要顏色 Color Selector
    Gradient Color 設定第二段落的漸變顏色 Color Selector
    Bar Opacity % 設定第二段落的不透明度(0-100%) Number Editor
    Corner Radius 設定第二段落的圓角半徑 Number Editor
    Border Color 設定第二段落邊框的顏色 Color Selector
    Border Width 設定第二段落邊框的寬度 Number Editor
    Shadow Color 設定第二段落陰影的顏色 Color Selector
    Shadow Width 設定第二段落陰影的寬度 Number Editor

    Main Color 設定第三段落的主要顏色 Color Selector
    Gradient Color 設定第三段落的漸變顏色 Color Selector
    Bar Opacity % 設定第三段落的不透明度(0-100%) Number Editor

    Line Padding Setting

    屬性名稱 屬性功能描述 操作型態
    Padding Top/Bottom 調整進度條和背景的上下邊距，最大最小值隨Widget高度而改變 Number Editor
    Padding Left/Right 調整進度條和背景的左右邊距，最大最小值隨Widget寬度而改變 Number Editor

    Gradient Color 設定滑鈕的漸變顏色 Color Selector

    Border Color 設定滑鈕邊框的顏色 Color Selector
    Border Width 設定滑鈕邊框的寬度 Number Editor
    Shadow Color 設定滑鈕陰影的顏色 Color Selector
    Shadow Width 設定滑鈕陰影的寬度 Number Editor
