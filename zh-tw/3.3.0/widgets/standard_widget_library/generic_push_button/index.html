    2. Standard Widget Library
    3. Generic Push Button

    有段按鈕是一個Input Widget，主要作為一個無內部狀態的按鈕。

    按下以及釋放按鈕可以對應到不同的顯示圖像以及文字。

    有段按鈕提供所有按鈕能觸發的事件，包括點擊、按下、釋放、短按、長按和持續按等動作，所有動作皆會產生事件並可以對應到頁面切換、暫存器數值改變或是執行使用者撰寫的Blockly程式。使用者可精細控制不同的觸發做不同的動作。

    Generic Push Button

    屬性名稱 屬性功能描述 操作型態
    Name Widget實體名稱 Name

    Visibility Control

    屬性名稱 屬性功能描述 操作型態
    Binding Hidden Control Register 決定是否要綁定暫存器，藉由暫存器內容觸發隱藏Widget Checkbox
    Hidden Control Register 透過改變暫存器值決定顯示或不顯示Widget Register Selector

    Image Setting

    屬性名稱 屬性功能描述 操作型態
    Mask Shape 決定遮罩形狀，分為橢圓及矩 Dropdown List

    Released State Text Setting

    屬性名稱 屬性功能描述 操作型態
    Display Text 決定放開按鈕時是否要顯示文字 Checkbox
    Text As Mask 決定放開按鈕時是否以文字為遮罩 Checkbox
    Text 顯示放開按鈕時的文字內容 Plain Text
    Font 設定放開按鈕時文字的字型樣式、字體大小、粗細、斜體及行距 Font Selector
    Color 設定放開按鈕時文字的顯示顏色 Color Selector
    Alignment 設定放開按鈕時文字的對齊方式 Alignment Selector
    Blending Mode 決定放開按鈕時文字的顏色與背景圖片的混色，詳情參考Blending Selector
    文字為遮罩時此功能無效    		 Blending Selector
    Mask Color 設定放開按鈕時的遮罩顏色 Color Selector

    Click Event Setting

    屬性名稱 屬性功能描述 操作型態

    Enable Register Operation 當按鈕被點擊時，根據Binding Register以及Operation Mode的內容對暫存器進行操作 Checkbox
    Binding Register 決定要改變的暫存器 Register Selector
    Operation Mode 當按鈕被點擊時，此處設定的行為會執行，詳情參考Register Operation Setting Register Operation Setting
    Enable Callback Function 當按鈕被點擊時，根據Behavior Programming設計的Callback Function進行操作 Checkbox
    Behavior Programming 當按鈕被點擊時，此處設定的行為會執行，詳情參考Blockly Editor Blockly Editor
