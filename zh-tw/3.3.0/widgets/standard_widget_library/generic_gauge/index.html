通用型儀表為輸出元件。

提供高效能省資源的數字表指針顯示，刻度為左右對稱，並可設定漸層顏色。此元件以直接繪製方式繪製指針，適合用於需要大量圓形指針並需要快速反應數值刻度的頁面。此元件具透明底色，可以疊在任何背景上增加顯示辨識度。

如欲快速入門，可以觀看博橡教育影片，幫助了解通用型儀表
