      1. Widgets
      2. Standard Widget Library
      3. Generic Button

      有段按鈕是一個Input Widget，主要作為一個無內部狀態的按鈕。

      為高效能以及節省資源的最佳選擇，僅提供特定字型尺寸的英文顯示以及漸層填滿與邊框顏色尺寸等風格設定。不提供載入圖像功能。

      有段按鈕提供所有按鈕能觸發的事件，包括點擊、按下、釋放、短按、長按和持續按等動作，所有動作皆會產生事件並可以對應到頁面切換、暫存器數值改變或是執行使用者撰寫的Blockly程式。使用者可精細控制不同的觸發做不同的動作。
