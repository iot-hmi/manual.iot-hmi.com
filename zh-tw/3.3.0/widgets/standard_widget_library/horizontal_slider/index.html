    水平圖像進度條可作為Output以及Input Widget，將Knob開啟時進度條為Input Widget，否則則為Output Widget。

    水平圖像進度條的背景、進度條、進度條底色和Knob等，皆可分別載入不同尺寸的圖形，使用者可以根據需求，製作不同風格及形狀的進度條，適合使用於需要顯示特別指示訊息的進度條，譬如電池存量或液體槽的液體高度等。

    Properties

    Horizontal Slider

    屬性名稱 屬性功能描述 操作型態
    Name Widget實體名稱 Name

    Visibility Control

    屬性名稱 屬性功能描述 操作型態
    Binding Hidden Control Register 決定是否要綁定暫存器，藉由暫存器內容觸發隱藏Widget Checkbox
    Hidden Control Register 透過改變暫存器值決定顯示或不顯示Widget Register Selector

    Background Image Setting

    屬性名稱 屬性功能描述 操作型態
    Show Image 決定是否要顯示背景圖片 Checkbox
    Show Origin Line 決定是否要顯示背景圖片的輔助線
    輔助線顯示Widget外框及中心點    		 Checkbox
    Load Image 從檔案系統選擇作為背景的圖片 File Open Dialog Box
    Edit Image 編輯滑桿的背景圖片 Image Editor Dialog Box
    Paste Image 貼上剪貼簿中的圖片作為滑桿的背景圖片  

    Liquid Column Image Setting

    屬性名稱 屬性功能描述 操作型態
    Show Image 決定是否要顯示水銀柱圖片，水銀柱隨著暫存器值改變 Checkbox
    Show Handler 決定是否要顯示水銀柱的調整圖示
    利用調整圖示調整水銀柱的位置，以及水銀柱的寬度    		 Checkbox
    Liquid Column Top Left 設定水銀柱左上的坐標
    水銀柱使用Show Handler時有效    		 Alignment Selector
    Liquid Column Bottom Right 設定水銀柱右下的坐標
    水銀柱使用Show Handler時有效    		 Alignment Selector
    Load Image 從檔案系統選擇作為水銀柱的圖片 File Open Dialog Box
    Edit Image 編輯滑桿的水銀柱圖片 Image Editor Dialog Box
    Paste Image 貼上剪貼簿中的圖片作為滑桿的水銀柱圖片  

    Knob Image Setting

    屬性名稱 屬性功能描述 操作型態
    Show Image 決定是否要顯示滑鈕圖片，可以藉由滑鈕改變暫存器的值 Checkbox
    Show Handler 決定是否要顯示滑鈕的調整圖示
    利用調整圖示調整滑鈕的開始與結束位置，以及滑鈕的大小    		 Checkbox
    Knob Top Left 設定滑鈕左上的坐標
    滑鈕使用Show Handler時有效    		 Alignment Selector
    Knob Bottom Right 設定滑鈕右下的坐標
    滑鈕使用Show Handler時有效    		 Alignment Selector
    Load Image 從檔案系統選擇作為滑鈕的圖片 File Open Dialog Box
    Edit Image 編輯滑桿的滑鈕圖片 Image Editor Dialog Box
    Paste Image 貼上剪貼簿中的圖片作為滑桿的滑鈕圖片  

    Bore Image Setting

    屬性名稱 屬性功能描述 操作型態
    Show Image 決定是否要顯示毛細管圖片 Checkbox
    Show Handler 決定是否要顯示毛細管的輔助圖示
    利用輔助圖示調整毛細管的位置，以及毛細管的寬度    		 Checkbox
    Bore Top Left 設定毛細管左上的坐標
    毛細管使用Show Handler時有效    		 Alignment Selector
    Bore Bottom Right 設定毛細管左上的坐標
    毛細管使用Show Handler時有效    		 Alignment Selector
    Load Image 從檔案系統選擇作為毛細管的圖片 File Open Dialog Box
    Edit Image 編輯滑桿的毛細管圖片 Image Editor Dialog Box
    Paste Image 貼上剪貼簿中的圖片作為滑桿的毛細管圖片  

    Scale Setting

    屬性名稱 屬性功能描述 操作型態
    Display Scale 決定是否要顯示刻度 Checkbox
    Select Tick Head Style 設定刻度是否要圓頭矩形 Dropdown List
    Scale as Mask 決定是否以刻度為遮罩
    如果選擇以刻度為遮罩，刻度會鏤空透出後面的顏色    		 Checkbox
    Mark Start Location 設定刻度開始位置坐標
    Display Scale時有效    		 Alignment Selector
    Mark End Location 設定刻度結束位置坐標
    Display Scale時有效    		 Alignment Selector
    Tick Granularity 設定刻度的分格數 Number Editor
    Tick Color 設定刻度的顏色 Color Selector
    Tick Style 設定刻度的框線風格 Frame Selector
    Display Sub Tick 決定是否要顯示子刻度 Checkbox
    Sub Tick Granularity 設定子刻度的分格數 Number Editor
    Sub Tick Color 設定子刻度的顏色 Color Selector
    Sub Tick Style 設定子刻度的框線風格 Frame Selector
    Sub Tick Length as % 設定子刻度的長度佔刻度長度的百分比(1-100%) Number Editor
    Display Number 決定是否要顯示刻度數字 Checkbox
    Start Number 設定刻度開始的數值 Number Editor
    End Number 設定刻度結束的數值 Number Editor
    Alignment 設定刻度數字的開始在刻度的開始、中間或者結束 Alignment Selector
    Number Font 設定刻度數字的字型樣式、字體大小、粗細、斜體及行距 Font Selector
    Number Color 設定刻度數字的顏色 Color Selector

    Knob Behavior Setting

    屬性名稱 屬性功能描述 操作型態
    Minimum Data Value 最小資料值，對應到滑桿開始 Number Editor
    Maximum Data Value 最大資料值，對應到滑桿結束 Number Editor
    Reversed Value Mapping 決定是否要反向對應數值
    Minimum Data Value對應到滑桿結束，Maximum Data Value對應到滑桿開始    		 Checkbox
    Value Binding Register 如果綁定的暫存器內容等於Minimum Data Value，滑桿顯示在開始位置
    如果綁定的暫存器內容等於Maximum Data Value，滑桿顯示在結束位置    		 Register Selector

    Preview

    屬性名稱 屬性功能描述 操作型態
    Animate 預覽滑桿隨著暫存器內容而變化

    屬性名稱 屬性功能描述 操作型態
